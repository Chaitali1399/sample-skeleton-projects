About

Tired of setting up the framework your project? Do you forget where to put configurations? Confused what libraries are required to get the framework up and running? Sample-skeleton-projects is all about saving time.

Wide variety

A good range of Java based frameworks have been covered till today and we are slowly and gradually expanding. Upcoming frameworks have already been marked and under review. To get an idea of what to expect, take a look at the project's issues page.

Samples included

Self explanatory samples have been included in each of the implemented framework with easy to understand javadocs. Comments have been added where-ever we felt necessary to explain something in detail.

Easy to setup

Ease of setup is our top most priority. Each project has been carefully setup in such a way that anyone who is familiar with the maven build tool can easily work his way through the project hierarchy. Each project has been implented following the standard directory layout.

Spark

A micro framework for creating web applications in Java 8 with minimal effort.

Spark Framework is a simple and lightweight Java web framework built for rapid development. Spark was originally inspired by the web framework Sinatra, but it's intention isn't to compete with Sinatra, or other similar web frameworks in different languages. Sparks intention is to provide a pure Java alternative for developers that want to (or are required to), develop their web application in Java. Spark is built around Java 8's lambda philosophy, which makes a typical Spark application a lot less verbose than most application written in other Java web frameworks.
